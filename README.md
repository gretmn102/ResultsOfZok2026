# Итоги ЗОКа 2026

![ResultsOfZok2026](https://gretmn102.github.io/ResultsOfZok2026/ResultsOfZok2026.png)

На [конкурсе](https://zok.cx/games/) участвовала 31-а игра. Мы заняли следующие места:

[Укушенный](https://zok.cx/game/3460/) — 18 место за... Непонятно за что. Одни говорят за недостаточную вариативность (победителю ЗОКа это не помешало), другие — за вампира, не соответствующего вампирам. Я же Лапку всю дорогу ругал за слабо прописанного протагониста. Теперь она потешается надо мной.

[Психотленный](https://zok.cx/game/3447/) — 20 место за скуфское вторичное брюзжание, философию, туманные намёки (хотя всё старался разжевывать и подавать прямо в лоб) и за удобрение для подлинных шедевров ИФни (хотя это, скорее, плюс, но почему-то минус).

Остальные игры из картинки представлять не буду исключительно из вредности.
